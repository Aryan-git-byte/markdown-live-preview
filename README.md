# Markdown-live-viewer
A native desktop markdown live preview app that offers side by side markdown drafting & preview. this app is written in rust based on `egui`.

## Features:
- history saved in a json file
- live preview
- timestamp

## Installation
- either download it from releases or

```
git clone https://github.com/Aryan-git-byte/markdown-live-preview
cd markdown-live-preview
cargo run
```
